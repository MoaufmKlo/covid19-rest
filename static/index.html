COVID19 REST-API

This is a REST-API to get data about COVID19 (new corona-virus) cases. Data sources are
    openZH (for detailed figures about Switzerland)
    Johns Hopkins University (for figures about the world - not yet implemented).


The data can be filtered by country, area and date. Output is available as json or csv.

API-Documentation

Please refer to https://github.com/apfeuti/covid19-rest
